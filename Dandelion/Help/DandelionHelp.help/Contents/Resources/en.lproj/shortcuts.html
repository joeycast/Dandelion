Keyboard Shortcuts

Navigate Dandelion quickly using these keyboard shortcuts on Mac.

Navigation

Action Shortcut
Go to History ⌘ + 1
Go to Insights ⌘ + 2
Open Settings ⌘ + ,

Window

Action Shortcut
New Window ⌘ + N
Close Window ⌘ + W
Minimize ⌘ + M

Editing

Action Shortcut
Select All ⌘ + A

App

Action Shortcut
Hide Dandelion ⌘ + H
Hide Others ⌥ + ⌘ + H
Quit Dandelion ⌘ + Q

Symbol Key

Symbol Key
⌘ Command
⇧ Shift
⌥ Option
⌃ Control
